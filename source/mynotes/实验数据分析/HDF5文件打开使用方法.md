# 写在前面
HDF5, 大量(海量?)数据存储的一种解决方案. HDF的全称是Hiearchical Data Format
HDF5 文件一般以 `.h5` 或者 `.hdf5` 作为后缀名，需要专门的软件才能打开预览文件的内容
HDF5 文件结构中有 2 primary objects: Groups 和 Datasets。
